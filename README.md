## Chơi game
1. Truy cập https://ndmt9102.itch.io/spaceship-invaders
2. Nhấn vào nút Run game.

### Cách điều khiển nhân vật:

Chuột trái - Bắn

Nhấn con lăn chuột - Đổi vũ khí

P - Dừng game, truy cập cửa hàng.

Q, W, E, R, T - Kỹ năng


### Cheats:

U - Thêm mạng

N - Bỏ qua màn chơi

Chuột phải - Nâng cấp vũ khí ngay lập tức
